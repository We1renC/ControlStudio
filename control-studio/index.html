Static DC Decoupler

            
計算 W = G(0)⁻¹ 使 DC 增益變對角。套用後 channel grid 應呈對角化，重算 RGA 應趨近單位矩陣。

            
            


            
            

              
Frequency-Point Decoupler W(jωc)

              

                在指定 crossover 頻率 ωc 計算 W(jωc) = G(jωc)⁻¹。檢查 off-diagonal residual 是否夠小作為 dynamic decoupling 的可行性指標。

Hankel 奇異值

          

            模型複雜度指標：Hankel 奇異值越小，代表對應狀態對輸入→輸出貢獻越小，可安全省略。
          

          
          

            

            

          

        


        
        

          
Gramian / SVD 詳情

          

            以截斷脈衝響應法近似可控性 Wc 與可觀性 Wo Gramian，顯示對角元素、條件數及
            Hankel 奇異值 σᵢ = √(λᵢ(Wc)·λᵢ(Wo))，條目越小越可約化。
          

          
          

        


        

          
SISO Robustness: Sensitivity Functions

          

            計算 |S(jω)|, |T(jω)|, |KS(jω)| 三條 sensitivity Bode。Peak |S|, |T| 對應 robust margin：peak < 1.8 = low risk，1.8-2.5 = medium，> 2.5 = high。

            Loop transfer L = C·G（current PID + plant）。SISO only。
          

          

            

              

                
                
              

              

                
                
              

            

            
            

            


            

            
Gain / Phase Uncertainty Sweep

            

              對 L(jω) 套 multiplicative 不確定性 k·e^(−jθ)，計算最壞情況 |S|/|T|/|KS| envelope。輸入 gain 範圍 ±% 與 phase 範圍 ±°；samples 為每邊取樣數（含 0）。

            計算 MIMO plant 傳遞矩陣的 `‖G‖∞`（最大奇異值頻率曲線峰值），用來看最壞輸入方向的最大增益。
          

          
          

          

        


        

          
H∞ Mixed-Sensitivity PID Tuning

          

            設定 W1（performance，低頻）和 W3（robustness，高頻）權重，在 PID 參數空間最小化 ‖[W1·S; W2·KS; W3·T]‖∞。
            按 Nelder-Mead 在 PID 空間搜尋。
