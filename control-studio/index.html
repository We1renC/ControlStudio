Singular Value Bode
Static Decoupler

計算 W = G(0)⁻¹ 使 DC 增益變對角。套用後 channel grid 應呈對角化，重算 RGA 應趨近單位矩陣。

Dynamic Decoupler @ ω

在指定 crossover 頻率 ωc 計算 W(jωc) = G(jωc)⁻¹。檢查 off-diagonal residual 是否夠小作為 dynamic decoupling 的可行性指標。

MIMO LQR

求解 CARE 得到 m×n 增益矩陣 K。Q 大→狀態收斂快；R 大→控制力小。

MPC (Discrete, Receding-Horizon)

有限預測 horizon 的 LQR-MPC baseline。先把 continuous plant 用 ZOH 離散化，再用 receding-horizon 計算 u₀。
SISO 模式自動用 plant 轉 controllable canonical SS；MIMO 模式直接用 mimoPlant ABCD。

Input Box Constraints

限制每步 u 在 [u_min, u_max] 範圍內。使用 Hildreth 座標下降法求解 condensed QP。

Robust Sensitivity (S / T / KS)

計算 |S(jω)|, |T(jω)|, |KS(jω)| 三條 sensitivity Bode。Peak |S|, |T| 對應 robust margin：peak < 1.8 = low risk，1.8-2.5 = medium，> 2.5 = high。
Loop transfer L = C·G（current PID + plant）。SISO only。

MIMO H∞ Norm ‖G‖∞

MIMO 模式下計算 plant 傳遞矩陣的 H∞ 範數（最大奇異值頻率曲線峰值）。反映最壞輸入方向的最大增益。

Gain / Phase Uncertainty Sweep

對 L(jω) 套 multiplicative 不確定性 k·e^(−jθ)，計算最壞情況 |S|/|T|/|KS| envelope。輸入 gain 範圍 ±% 與 phase 範圍 ±°；samples 為每邊取樣數（含 0）。

H∞ Mixed-Sensitivity Synthesis

設定 W1（performance，低頻）和 W3（robustness，高頻）權重，最小化 ‖[W1·S; W2·KS; W3·T]‖∞。 按 Nelder-Mead 在 PID 空間搜尋。

GA Auto-Tuner

遺傳演算法搜尋最佳 PID 參數，目標為 overshoot + settling + IAE 的加權和。

Phase Portrait (2D)

僅 MIMO 模式且 n=2（2 個狀態變數）時可繪製。顯示向量場與軌跡。
