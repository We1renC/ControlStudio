共通落地流程

  1. 先在 build.nvidia.com 選模型，確認 Free Endpoint、Partner Endpoint 或 Download Available。
  2. 原型期用雲端 API endpoint；需要資料落地、低延遲或成本控管時改用可下載 NIM 自部署。
  3. 依任務建立最小資料集，測準確率、延遲、成本、失敗案例與安全風險。
  4. 將模型包成內部 service，統一處理 API key、重試、timeout、logging、版本。
  5. 上線時加監控、人工覆核與回饋資料，定期重測。
